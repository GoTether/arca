What is Arca Tethr?

Scan an NFC tag to jump straight to a room or container and adjust inventory with a tap. Photos and notes are optional. You can also open manually below.

Open by ID
Enter an Arca and a container ID to view.
How tags work
  • Write an NFC tag to this page: …/index.html?id=YOUR_TAG
  • If unassigned, you’ll enter an Arca ID (new or existing), pick Room/Container, and name it.
  • Future scans deep-link straight to that node’s inventory.
