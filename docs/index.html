Open an Arca

Scan an NFC/QR tag that appends an id (or arcaId) to the URL, or enter the Arca ID manually.

Tip: Program tags like …/index.html?id=PANTRY-A1 or …/display.html?arcaId=PANTRY-A1.

What is Arca?

A lightweight, tag-driven inventory for rooms, shelves, bins, or any container. Tap a tag to jump straight to that Arca, add items with photos/notes, and adjust quantities fast.
