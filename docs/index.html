What is Arca Tethr?

Scan an NFC tag to jump straight to a room or container and adjust inventory with a tap. Photos and notes are optional. You can also open manually below.

How tags work
  • Write an NFC tag with a URL to this page: …/index.html?id=YOUR_TAG
  • If the tag isn’t assigned, you’ll create a brand-new Arca + container for it.
  • Future scans deep-link straight to that inventory.
