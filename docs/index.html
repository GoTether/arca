What is Arca Tethr?

Arca Tethr is the fast, NFC-friendly way to track what's inside rooms, containers, and drawers. Scan a tag → jump straight to that container → add or subtract items with a tap. Photos and notes are optional.

Open by ID
Manually enter an Arca and container to view.
How tags work
  • Write an NFC tag with a URL to this page: …/index.html?id=YOUR_TAG
  • First scan assigns the tag to a specific room or container.
  • Future scans deep-link straight to that inventory.
